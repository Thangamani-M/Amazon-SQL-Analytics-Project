# Amazon-SQL-Analytics-Project
MySQL-based Amazon Fresh analytics project featuring relational database design, ER modeling, normalization (3NF), DDL/DML operations, constraints, joins, subqueries, window functions, and business insights. The project analyzes customers, products, suppliers, orders, and reviews to identify top-selling categories, high-value customers, revenue trends, supplier performance, and customer behavior, demonstrating practical SQL and database management skills for real-world e-commerce analytics.
